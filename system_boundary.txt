1.
Что будет включено:
Консольный интерфейс ввода/вывода
Игровая логика
Учёт статистики
Механизмы бонусов

Чего точно не будет:
Графический интерфейс
Многопользовательский режим
Рейтинг игроков

2.
Подситема обработки ввода и вывода
Подситема отвечающая за работу с игровым полем
Подсистема отвечающая за игровую локигу
Подсистему отвечающая за статистику
Подсистема бонусов и их применения

3.
"Игрок" - человек, который вводит в консоль данные для взаимодействие с программой
"Игровое поле" — сетка размером 8x8, заполненная случайными элементами из 5 типов (например, буквы A, B, C, D, E)
"Комбинация" — цепочка из 3+ одинаковых фишек подряд
"Очки" — результативность игрока за комбинации и бонусы

4.
Вывод в консоль игрового поля.
Интерактивный ввод для перестановки элементов.
Перестановка элементов по вводу игрока.
Учет статистики игрока.
Бонусная система.

5.
Java Standard Library
JUnit
